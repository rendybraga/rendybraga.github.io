Deskirpsi Tentang Saya

Assalamualaikum WR.WB, Perkenalkan nama saya Randy panggil aja Rendy, saya lahir di makassar 21 Agustus 2003. Orang tua saya asli suku bugis Makassar, diumur saya yg ke 2 tahun saya pergi ke Jakarta bersama orang tua saya agar saya bisa berpendidikan di Jakarta. Hobi saya dari dulu sering mendengarkan musik dan saya suka yg bergendre musik keras/metal, saya juga hobi dengan motor classic dan saya suka meng custom motor-motor lama menjadi motor retro/classic.

CIta-cita saya ingin membanggakan kedua orang tua saya, dan saya ingin menjadi orang yg sukses dimasa yg akan datang. Pendidikan saya yg pertama di SDN Kebon Melati 02 Pagi, yg ke-2 di SMPN 38 Jakrata, dan yg sekaran di SMKN 19 Jakarta.
